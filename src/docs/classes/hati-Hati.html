Hati.php : 19

Hati, a speedy PHP library. This class does all the magic under the hood. It gets the very first call from the server, before any code can execute. This prepares dependencies by setting a class loader.

It uses the configuration object to prepare the working environment properly. Please use hati.json file in order to customize your great HATI.

getConfigPath()

Hati.php : 133

Based on location, where the hati is being used, it tries to find the config path.

private static getConfigPath() : void

This helps Hati to figure out whether it is used for sub-project & load configuration appropriately.

It tries for 15 times, from current working directory to see if there is any folder called hati and there is a hati.json file exists. If so then Hati loads that as sub-project config file. However, on failure, Hati falls back to root project config file.
