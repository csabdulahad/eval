prompt()

CLI.php : 117

Asks the user for input.

public static prompt(string $field[, array<string|int, mixed>|string|null $options = null ]) : string

Usage:

// Takes any input $color = CLI::prompt('What is your favorite color?');

// Takes any input, but offers default $color = CLI::prompt('What is your favourite color?', 'white');

// Will validate options with the in_list rule and accept only if one of the list $color = CLI::prompt('What is your favourite color?', array('red','blue'));

// Do not provide options but requires a valid email $email = CLI::prompt('What is your favourite color?', null, ['red', 'grey']);

Parameters
$field : string

$options : array<string|int, mixed>|string|null = null
