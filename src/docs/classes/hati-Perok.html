in package
  • Application

Perok.php : 24

Perok is a wrapper class around PHP Mailer library. The class has singleton pattern implemented for providing a simple line API to the client code. It uses SMTP protocol

Protocol configuration for this such as email & password can be set using hati/hati.json. file.

Google SMTP can be used too. If the google account has two factor authentication turned on, then it needs a one time 16 digit password from the google account to be set in the config file.

Many powerful emailing feature such as attachment, HTML email, CID image in HTML, BCC are well supported by this class.
