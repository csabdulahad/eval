Templating in project sometimes are necessary to have. Hati provides a wonderful support for templating with Template library class. It is simple in nature yet very powerful in capabilities.

Any template file must have an extension of '.tlp.php' which clears up many confusion and makes it easier to mark that the file is meant to go through template engine.
