Fluent.php : 35

Fluent is wrapper class around PDO extension to allow simple, flawless easy access and manipulation of the database query operations. It uses singleton pattern to permits its instance. This also supports transaction operations on database using various methods such as beginTrans, rollback & commit.

Any data returning method assumes that a successful connection has made to the database and a query has already been executed. Currently it has three data returning methods such as:

  • : returns the array containing array of the result/rows
  • : returns first array of the data
  • : returns specific property of the datumArr

For better security and practice, it is recommended that call to any Fluent method should be inside try-catch block to hide the throwing error message or reactive to any error.

use()

Fluent.php : 116

Since Hati 5, Fluent can work with multiple database connection profiles.

public static use(string $dbProfile) : PDO|null

Database connections are specified by objects in the hati/db.json file where each object is identified by their profile names.

A db profile can have many database. Connection to each database is represented by id. Id naming has a strict convention. It should be composed of database profile name followed by colon, followed by the database name it connects to. For example, to represent a connection to database test2 on localhost, using root @ pass, the profile id would be: Example-Localhost:test2
{ "db_profiles": { "Example-Localhost": { "address": "localhost", "username": "root", "password": "pass", "db": ["test1","test2"] } } } Note: Use "hati/tool/dumpdb.php" tool to generate these profile ids automatically for your project's db.json file.

Parameters
$dbProfile : string

The db profile id

Return values
PDO|null —

The pdo connection object to the database
