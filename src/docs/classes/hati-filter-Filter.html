Filter
in package
  • Application

AbstractYes
Filter.php : 37

Filter class is very helpful and handy in situations where user inputs need to be filtered and sanitized. It has many methods which are of different capabilities.

All the methods are safe as they use both validation and sanitization.

They all returns the validated & sanitized input value upon successful filtering. To check if the data returned by one of filter functions was all ok, use method on the returned value. For example:

$email = Filter::email($_POST['email']); if (Filter::ok($email)) { echo 'Email accepted'; } else { echo 'Email invalid'; }

There is one special method which can directly work with any regular expression or some predefined pattern constants by Filter class. Predefined regular expression pattern for filtering input in various formats are provided. Those constants start with SAN_ name. For example: Filter::SAN_ANSD In the naming of these constants, they have meaning like regular expression.

  • A = Alphabets(including capital & small letters)
  • N = Numbers
  • S = Space
  • C = Comma
  • D = Dot

These patterns removes other characters from the input as specified.
