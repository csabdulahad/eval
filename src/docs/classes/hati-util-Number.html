Format method can take any input and tries to represent it into either money format or a leading zero, decimal placed number format. The number can be either signed or unsigned. It cleverly, add the currency signs before and in between the negative sign.

public static format(mixed $input[, string $sign = '' ][, bool $leadZero = false ][, int $place = 2 ][, bool $print = false ]) : string|null

Since it takes mixed as input type, there are many possibilities that it could fail to to format a number. It tries its best to format a number by forcefully casting the input into float when both normal integer and float filtering fail. It prints out 0-0 on encountering error while formatting the input.
