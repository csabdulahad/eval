Trunk extends RuntimeException
in package
  • Application

Trunk.php : 22

Trunk is the native error type for Hati library. In various parts of the library, Trunk is used to throw errors.

Any implementation of can use this Trunk class to report back to the requester with JSON. There are many static methods in this Trunk class to output various types of error which start with Trunk::errorXXX where XXX is the HTTP status code.
